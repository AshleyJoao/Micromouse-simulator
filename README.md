IEEE Micro-mouse simulator 

A pygame-based simulation of a micromouse robot navigating a maze using classic maze-solving algorithms. Watch the mouse explore, map, and solve the maze in real time



Preview :
8x8 grid maze with red walls
distance value displayed in every cell
purple trial showing the mouse's path
Orange circle re[resent the mouse
Green S (start) and Red T (target) labels



Features
Flood fill algorithm - compute shortest path using BFS
horizontal and vertical wall arrays for an 8×8 grid
Clean rendering — distance numbers, walls, grid lines, and mouse all layered clearly


Requirements 
Python 3.8+
pygame

Install pygame:
pip install pygame
